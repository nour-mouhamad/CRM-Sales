# CRM Sales — Cockpit de closing (IQ Project)

CRM interne pour closers vendant des formations financées CPF. Mono-fichier React (Babel standalone), zéro install : ouvrir `index.html` dans un navigateur.

## Pipeline
Lead à contacter · R1 à faire · R2 à faire · R3 à faire · À payer · Payé · CPF (Nour) · Prospect perdu

## Stack
HTML + React (JSX) + Hanken Grotesk / JetBrains Mono. Palette « Terracotta », mode clair/sombre. Tout mocké, transitions d'état réelles (drag & drop, appel + recap Granola, envoi lien paiement → payé, RDV closing & admission CPF).
